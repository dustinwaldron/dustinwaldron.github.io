NextJS Pokedex App

Background Years ago, I started my development journey learning javascript, NodeJS and React. My first project was, like others, building a Pokedex App using the PokeAPI. The architecture was pretty simple: The UI ran with vanilla javascript and React 15 The backend was a REST API build with NodeJS and Express The database was a flat json file that was basically the same schema as the PokeAPI contract The database was populated by a batch job that fetched the first generation of pokemon flowchart TD UI["React Pokedex UI"] API["Node.js Pokemon API"] DB["JSON Pokemon Database"] BATCH["Batch Job (Fetches Gen 1 Pokémon)"] UI --> API API --> DB BATCH --> DB Since then, there have been many new frameworks and enhancements made in the javascript community that would put my code then to shame. So, in honor of it being nearly 10 years since I did this humble project, I’ve decided to recreate this nostalgic Pokedex app! ...
